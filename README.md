# KIRIMIN DOCS

> This docs if for [Kirimin Messenger](https://kirimin.devanka.id).

[Kirimin Messenger](https://kirimin.devanka.id) was made for educational purposes only. Written in [TypeScript](https://www.typescriptlang.org) and uses [Node.js](https://nodejs.org), [Express](https://expressjs.com), [Corturn](https://github.com/coturn/coturn), [Nodemailer](https://nodemailer.com), [SASS](https://sass-lang.com), and [Webpack](https://webpack.js.org). Running on [Ubuntu](https://ubuntu.com/server) and uses [Git](https://git-scm.com) to control its version. All problems can be reported via email contact@devanka.id or direct message on [Discord](https://devanka.id/discord).

**Kirimin Docs** offers free API for developers who want to build/expand their own [Kirimin Messenger](https://kirimin.devanka.id) client on other devices.
